## 4. 错误码
| HTTP | code | 含义 | 说明 |
| --- | --- | --- | --- |
| 400 | 400 | Bad Request | 参数校验失败 |
| 401 | 401 | Unauthorized | 未认证或 Token 过期 |
| 403 | 403 | Forbidden | 无权限 |
| 404 | 404 | Not Found | 资源不存在 |
| 409 | 409 | Conflict | 状态冲突 |
| 500 | 500 | Server Error | 服务器异常 |
