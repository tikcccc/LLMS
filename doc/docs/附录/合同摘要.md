# 合同摘要（去价格版）

## 项目概述
- 目标：建设“地块作业管理系统（LLMS）”，提供 Web 管理端与移动端协同能力。
- 关键优势：直连香港政府 CSDI（Lands Department）官方地理数据，提升准确性并避免商业地图授权成本。

## 功能范围（Functional Scope）

### 1) Web Portal
- 交互式地图：支持 CSDI 底图（街道 / 地形），缩放、平移、搜索。
- 图层管理：政府图层与自定义图层开关，对比视图同时展示。
- 绘制与编辑：创建/编辑 Work Lot 多边形并维护状态信息。
- 任务模块：任务分配、看板视图、地图状态颜色预警（红/黄/绿）。
- 报表与查询：列表汇总与地图框选查询。

### 2) Mobile Application
- 地图与导航：CSDI 底图 + 缩放/平移/搜索。
- 任务列表：查看任务清单。
- GPS 核验：显示当前位置（仅参考）。
- 地块信息：查看 Work Lot 详情与状态。

### 3) 用户管理
- 管理员：系统设置与权限管理。
- 业务用户：创建 Work Lot、分配任务。

## 非功能范围（Non-functional）
- 可用性：Web 端响应式适配桌面/笔记本/平板。
- 安全：密码加密存储；通信使用 SSL/HTTPS。
- 数据准确性：基于 CSDI 官方数据。
- 性能：支持同时展示数百地块并保持流畅交互。

## 专业服务（Professional Services）
- 项目管理：提供项目负责人、周报与进度同步。
- 培训：系统管理员培训 + 管理者“Train the Trainer”。
- 文档：提供用户手册与管理员指南（PDF）。

## 里程碑与周期（Timeline）
- Week 1–2：项目搭建、核心架构、CSDI 接入。
- Week 3–5：图层与任务模块、移动端并行开发。
- Week 6–7：安全、QA 测试、文档与最终交付。
- 总周期：6–8 周。

## 条款与限制（Terms & Conditions）
- 部署位置：香港云服务器。
- 数据来源依赖：CSDI 官方门户可用性。
- App 上架：不公开上架，提供直接下载方式。
- 支持：上线后提供 30 天支持。
