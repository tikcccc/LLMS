# 术语表

本术语表统一定义 LLMS 项目中的核心概念与业务术语，确保各方理解一致。

## 核心实体

| 术语 | 英文 | 定义 | 备注 |
|------|------|------|------|
| 地段 | Land Lot | 地政总署（LandsD）定义的法律地界（Cadastral Boundaries），例如 D.D. 123 Lot 456 | 来自官方记录，作为参考图层（Reference Layer） |
| 作业地块 | Work Lot | 用户在系统地图上手动绘制的多边形，用来框出 Operator 的实际占用范围 | 系统的核心操作单元（Active Layer） |
| 营运者 | Operator | 实际在土地上的人或机构，包括 BU（Business Undertaking）或 Household（住户） | 例如："顺意厂房" |
| 任务 | Task | 需要在指定 Work Lot 上执行的工作项 | 例如："查证边界"、"协商"、"确认交地" |

## 业务术语

| 术语 | 英文 | 定义 | 备注 |
|------|------|------|------|
| BU | Business Undertaking | 业务经营者，指在地块上经营的商业实体 | 属于 Operator 的一种类型 |
| 住户 | Household | 在地块上居住的家庭单位 | 属于 Operator 的一种类型 |
| 收地 | Land Resumption | 政府收回土地的法律程序 | 项目的核心业务场景 |
| 交地 | Handover | 将已收回的土地移交给承建商 | Work Lot 的关键状态节点 |
| 清场 | Clearance | 清理地块上的建筑物、设施和人员 | 交地前的必要步骤 |

## 技术术语

| 术语 | 英文 | 定义 | 备注 |
|------|------|------|------|
| CSDI | Common Spatial Data Infrastructure | 香港政府共用空间数据基础设施 | 提供官方底图与地理数据 |
| HK80 | Hong Kong 1980 Grid System | 香港 1980 坐标系统（EPSG:2326） | 项目统一使用的坐标系 |
| 底图 | Base Map | 地图的背景图层，显示街道、地形等基础地理信息 | 来自 CSDI Topographic Map API |
| 图层 | Layer | 地图上可独立开关的数据层 | 包括政府图层、自定义图层等 |
| 多边形 | Polygon | 由多个坐标点连接形成的封闭区域 | 用于表示 Land Lot 和 Work Lot 的边界 |
| GPS 核验 | GPS Verification | 使用移动设备 GPS 确认现场位置 | 仅供参考，非测量级精度 |
| 吸附 | Snapping | 绘图时自动对齐到已有边界的功能 | 提高绘图精度和效率 |

## 状态定义

| 状态 | 英文 | 定义 | 颜色标识 |
|------|------|------|----------|
| 待处理 | Pending | 任务已创建但尚未开始 | 灰色 |
| 进行中 | In-Progress | 任务正在执行中 | 黄色 |
| 已交地 | Handover | 地块已移交给承建商 | 绿色 |
| 困难个案 | Difficult Case | 存在纠纷或其他阻碍因素的地块 | 红色 |
| 已完成 | Completed | 任务已完成 | 绿色 |
| 已逾期 | Overdue | 任务超过截止时间仍未完成 | 红色 |

## 角色定义

| 角色 | 英文 | 定义 | 主要职责 |
|------|------|------|----------|
| 系统管理员 | Site Admin | 负责维护 Land Lot 数据的管理人员 | 测量/IT 人员，可编辑 Land Lot |
| 现场主任 | Site Officer | 负责维护 Work Lot 和任务的前线管理人员 | 可编辑 Work Lot，分配任务 |
| 现场人员 | Field Staff | 执行任务的一线工作人员 | 移动端查看任务，更新状态 |
