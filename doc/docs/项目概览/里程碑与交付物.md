## 2. 排期与交付物清单

| 阶段 | 周次 | 主要交付物 | 验收标准 |
| --- | --- | --- | --- |
| 基础与原型 | Week 1-2 | 开发环境搭建完成<br>CSDI Topographic Map API 集成成功<br>HK80 坐标系注册和 TileGrid 配置<br>基础绘图功能（Draw Polygon）<br>可视化 Demo（预置 demo polygons） | 底图正常显示，署名信息完整<br>可绘制多边形并保存<br>刷新页面数据不丢失 |
| 核心功能开发 | Week 3-6 | Web 端完整功能（地图、图层、Work Lot、任务、报表）<br>Mobile 端基础功能（任务列表、GPS、地块查看）<br>角色权限控制<br>搜索与查询功能 | 所有用户故事验收通过<br>角色权限正确生效<br>移动端可正常使用 |
| 测试与交付 | Week 6-8 | UAT 测试完成<br>文档完整（用户手册、技术文档）<br>培训材料<br>部署到 C1 Site Server | UAT 通过率 ≥ 90%<br>文档完整且可读<br>用户培训后可独立操作 |
